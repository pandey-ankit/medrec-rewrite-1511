Preparing the Sample

  1. Set the samples environment:

    2. Change to the @MW_HOME/wlserver/samples/server directory and run the setExamplesEnv script.

  2. Set the WebLogic properties:

    3. Change to the @MEDREC_HOME/install/non-mt-single-server/ directory. Open the weblogic.properties file and update these five Administration Server properties: 

      admin.server.name
  admin.server.host
  admin.server.port
  admin.server.username
  admin.server.password


Installing the Sample

From the @MEDREC_HOME directory, run: 

  ant single.server.sample

After the installation completes, the MedRec welcome page will open automatically. The WebLogic domain is located at @MW_HOME/user_projects/domains/medrec.


Control Commands List

From the @MEDREC_HOME/install/non-mt-single-server directory, review the details of the build.xml file.

  1. Start the Adminstration Server:
    2.   ant startAdminServer
  2. Shutdown the Adminstration Server:
    3.   ant stopAdminServer
  3. Start Derby listening to the Adminstration Server:
    4.   ant derby.start
  4. Stop Derby listening to the Adminstration Server:
    5.   ant derby.stop
  5. Start all of the "start" commands listed above:
    6.   ant start
  6. Stop all of the "stop" commands listed above:
    7.   ant stop
